== Description ==

Corpex: A sleek and modern WordPress theme designed for corporate websites. With its intuitive interface, customizable options, and responsive design, Corpex is perfect for showcasing your business's professionalism and innovation. View the demo of Corpex Pro https://www.nayrathemes.com/demo/pro/?theme=corpex


== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
